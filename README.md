# 소개

- ### 프로젝트 소개

추천수가 일정 수를 넘어선 글만 보여주는 ***BEST게시판***과 ***RESTful 다중 게시판***,  
***AJAX***를 이용한 ***비동기 추천***과 ***게시판/게시물 북마크***,  
***첨부파일 업/다운로드***, 회원간 ***쪽지*** 기능 등을 갖춘 ***커뮤니티 사이트***입니다.
  
  
- ### 제가 맡은 역할

***단독 개발***입니다.
